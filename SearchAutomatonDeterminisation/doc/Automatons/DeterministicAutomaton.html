Automatons::DeterministicAutomaton

This class represents a deterministic search automaton. It is built from NonDeterministicAutomaton and provides methods for continuous state changing. It also uses a listener, which is notified every time, automaton reaches some of its final states.

153:             #if that node is final, the new deterministic node will be final too.

154:             if(ndNode.finalPattern != nil)

156:             end

157:             #get transitions for current alphabet symbol 'a'
